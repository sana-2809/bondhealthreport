{\Large \textbf{ABSTRACT}}
\end{center}

Automated Diagram Digitization is a modern web--based application designed to convert Automated Diagram Digitization is a web-based platform that converts flowchart images into interactive, editable digital diagrams by leveraging artificial intelligence for automatic shape recognition, text extraction, and connection detection. Using Google's Gemini AI via a custom-engineered prompt, the system identifies four standard flowchart shapes -- process (rectangle), decision (diamond), start/end (oval), and input/output (parallelogram) -- and structures the extracted information as React Flow-compatible JSON. Three input methods are supported: image upload, text-to-flowchart generation from natural language, and direct JSON import. The interactive editor provides undo/redo, auto-layout, edge labeling, text and color styling, and export to PNG, SVG, and JSON. The backend is implemented in FastAPI with PostgreSQL for persistent storage, and the frontend in React with Vite. Robust error handling, custom prompt engineering, and comprehensive RESTful endpoints collectively deliver a production-ready solution for students, educators, developers, and business professionals who require efficient flowchart digitization and editing.
